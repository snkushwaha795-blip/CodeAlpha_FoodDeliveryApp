# Food Delivery App – Mobile Redesign

## Overview
This project is a UX-driven mobile redesign of a food delivery application. 
The goal was to improve usability, simplify navigation, and enhance the 
overall ordering and checkout experience.

## Goals of the Redesign
- Improve user flow and reduce friction
- Enhance visual hierarchy
- Create consistent UI components
- Streamline checkout process

## Tools Used
- Figma
- Auto Layout
- Components & Variants
- Prototyping

## Key Features Designed
- Onboarding Flow
- Login / Signup
- Home Screen
- Restaurant Details
- Cart & Checkout
- Payment Flow
- Order Confirmation
